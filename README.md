# DevHub

DevHub is a clean, straight-up platform for developers who want to build in public, share their work, ask questions, and showcase their learning journey. No drama â€” just a space where devs can vibe, grow, and connect.

**LINK:- https://devvhubb.vercel.app/**

## Features
- Authentication & Profiles: JWT auth, profile updates, add skills/projects, protected routes.
- Posts & Feed: Create posts, like/unlike, comment, all with real-time updates.
- Q&A System: Ask questions, comment, vote, with instant live syncing.
- Real-Time Updates: Live posts, comments, and votes powered by WebSockets.
- API Documentation: Fully documented backend APIs using Swagger -> Access it at: https://devhub-server-zugl.onrender.com/api-docs/
