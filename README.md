# DevHub

DevHub is a clean, straight-up platform for developers who want to build in public, share their work, ask questions, and show what theyâ€™re learning. No drama, just a space where devs can vibe, grow, and put their journey out there.

## Features
- Build in public by posting your work, updates, and progress
- Create your own profile with name, username, bio, skills, and projects
- Ask a dev by posting questions with description and tags
- Explore what other devs are building or struggling with
- Minimal, fast, and easy-to-use interface
